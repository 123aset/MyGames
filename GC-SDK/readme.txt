В архиве 3 папки:
1) GCServer - запуск UI, выбор карты и кол-во игроков
2) Bot - сама реализация бота. Там есть классы Cell.java, Direction.java, PlayerThread.java нужны для работы клиента, в их исходники можете в принципе не лезть.
Самое главное создать свой класс, сэкстендить от класса PlayerThread и прописать метод move(). 
Можете посмотреть в качестве примера DummyBot.java
3) GCMapDesigner - если сами захотите создавать карты

Шаги для запуска:
1) Запустить сервер. Для этого в папке уже есть Launcher.bat, который запускает карту map.txt с одним игроком.
java -jar GCServer.jar [путь к карте] [кол-во игроков]
